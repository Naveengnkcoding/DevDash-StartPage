# 🚀 Browser-StartPage (Developers): The Ultimate Developer Start Page

A lightweight, browser Startpage designed for developers. It runs entirely from a local HTML file, giving you total control over your workspace with 100% privacy.

## ⚡ Key Features

1. **Dev Bookmarks:** Custom quick-links that automatically open in a **new tab**.
2. **Multi-Search Engine (v2+):** A built-in quick search bar supporting multiple search engines (all opening in new tabs).
3. **Multi-Slot Clipboard Cache:** Dedicated slots to save frequent usernames, app IDs, or text strings directly to your browser cache—better than OS history!
4. **Attractive Wallpapers:** Curated backgrounds to keep your workspace looking fresh.
5. **🕵️‍♂️ Hidden Secrets:** Built-in secret windows and easter eggs. *If you're smart, you'll find them.*

## ⚠️ Important Note: Backup Your Data!

> 🚨 **CRITICAL:** This Startpage saves everything inside your browser's **LocalStorage**. If you clear your browser cookies/cache, your data will be wiped. 
> 
> **Always export and backup your Setup configuration as a JSON file** to ensure you never lose your data!
