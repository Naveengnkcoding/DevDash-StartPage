Set as Your Start Page

Configure your browser to open Startpage on every new tab and startup. It's simple and takes under a minute.

1

Download the Release

Grab the latest .html file from the Alpha or Beta series above. Save it to a local folder on your machine.

2

Set as Startup Page

In Chrome/Edge/Firefox settings, set your startup pages to open the local file. Use file:///path/to/startpage.html or host it locally.

3

Use an Extension (Optional)

For new-tab replacement, use extensions like New Tab Override (Firefox) or configure Chrome's new tab redirect to point to your local file.

4

Customize & Enjoy

Edit the JSON config at the top of the file to add your name, links, projects, and cities. No build step required — it's all plain HTML, CSS, and JS.
